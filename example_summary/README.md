# Example Intelligence Report

This folder contains a sample **FreightLens Intelligence Report** (PDF) generated by the app.

- **`freightlens_intelligence_report_example.pdf`** – Example multi-section report (executive summary, shipment/carrier/route/financial/fraud analytics, AI insights) for reference and demos.

Generate your own report from the dashboard via **Navigation → Generate Intelligence Report** after uploading LR, POD, and Invoice datasets.
